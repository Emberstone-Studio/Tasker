Tasker

A local-first kanban board for AI agent tasks. Assign tasks to agents, configure models, and let tasker.js handle execution. State is stored locally and embedded back into this HTML file on save.

1
Make sure Node.js is installed
node --version
2
Run Tasker — it opens this page automatically
node tasker.js
3
Configure a model in Settings (gear icon, top-right)
4
Create a task, set it to Ready, and hit Execute
